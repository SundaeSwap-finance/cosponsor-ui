### Key Features Delivered

1. **Browsing** - View and filter governance proposals
2. **Contributions** - Deposit ADA to sponsor proposals, receive gADA tokens
3. **Refunds** - Withdraw ADA by burning gADA tokens
4. **Wallet Integration** - Connect Cardano wallets (CIP-30)

### Technical Stack

- **Frontend:** React 19, Vite, TypeScript, Tailwind CSS
- **Blockchain:** Cardano (Preview Testnet → Mainnet)
- **Smart Contracts:** Aiken (PlutusV3)
- **SDK:** Blaze Cardano SDK
- **Provider:** Blockfrost
